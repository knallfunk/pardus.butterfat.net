Pardus Butter Stocker

This is a Firefox extension to help traders in the game Pardus. The extension is heavily based on Astagor's PSH. When you install it, there will be a new option under the Tools menu.

The data recorded by the extension is available here on the PBS web site

Insane props to Astagor. Back-end coding by Satyap. This is part of the Pardus Butterfat project.
